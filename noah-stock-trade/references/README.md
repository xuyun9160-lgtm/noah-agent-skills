- `current-availability.md`：当前 trade 能力状态矩阵
- `order-status-mapping.md`：订单状态码与用户态解释映射
- `output-policy.md`：订单 / 成交 / 费用 / 可买可卖结果的输出策略
- `usage-guide.md`：当前只读交易能力的使用说明与示例场景
