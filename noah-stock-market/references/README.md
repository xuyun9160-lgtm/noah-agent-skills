# References Index

## Purpose
本目录用于沉淀 `noah-stock-market` 模块的能力边界、安装前提、路由规则、输出策略、已知限制和示例问法。

## Documents
- `auth-and-preflight.md`：鉴权与前置检查说明
- `current-availability.md`：当前能力状态矩阵（Ready / Partial / Unsupported）
- `current-status-summary.md`：当前阶段能力状态摘要
- `edge-cases.md`：已知边界场景与踩坑记录
- `error-catalog.md`：常见错误类型、原因与恢复建议
- `install-troubleshooting.md`：安装与接入排障指南
- `known-limitations.md`：当前环境限制与未开放能力说明
- `market-examples.md`：示例问法与典型查询场景
- `name-resolution.md`：股票名称 / 代码解析规则
- `output-policy.md`：最终用户输出策略与风格规则
- `response-modes.md`：开发模式与股票帮手模式说明
- `routing-and-intents.md`：早期意图与路由说明
- `routing-spec.md`：当前正式路由规格说明
- `usage-guide.md`：使用说明与安装后建议路径
