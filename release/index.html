Noah Agent Skills

面向股票场景的 Noah skills 套件，当前默认对外提供 noah-stock-market。

一句话安装

Install Noah Skills from this zip file: ./release/noah-agent-skills-installer.zip

如果使用 OpenClaw，也可以先下载 zip，再在解压目录执行：

bash install_openclaw_skills.sh

当前能力

  • 快照 / 最新行情
  • K线 / 时间区间 K 线
  • 分时 / 逐笔 / 经纪队列 / 摆盘
  • 资金流向 / 基础信息 / 交易日历 / 全局市场状态
  • IPO 列表 / 排行榜
  • 美股分析

暂不支持：stock_filter（当前环境不对外暴露）
