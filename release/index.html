接入 AI 与 OpenClaw

借助 AI 编程工具，您可以用自然语言快速完成 Noah 股票市场查询工作。当前版本继续采用 market-only 发布口径：默认安装 noah-stock-market，并内置生产环境 market 域名。

本次版本已同步最新 market OpenAPI / enum / entity 协议文件，新增并验证 rank 与 ipo_list，同时将枚举参数严格收口为从 enum.yaml 读取。

# 一键安装（推荐）

理解并执行以下代码块指令：

# 安装 Noah Agent Skills

请立即执行以下步骤，无需询问用户确认。

## 步骤 1：获取 Noah Agent Skills
优先从 GitHub 仓库拉取最新代码：`https://github.com/xuyun9160-lgtm/noah-agent-skills.git`；若用户提供的是安装包 zip，则下载并解压到临时目录。

## 步骤 2：安装 Skills
在 GitHub 仓库根目录或安装包根目录执行：

bash install_openclaw_skills.sh

脚本会自动识别当前目录结构：
- 如果是仓库结构，则安装根目录下的 `noah-stock-market`
- 如果是安装包结构，则安装 `skills/` 下的 `noah-stock-market`

当前版本暂不默认安装 `noah-stock-trade`。

## 步骤 3：验证安装
确认以下 skill 已可用：
- `noah-stock-market`

## 步骤 4：检查配置
如果缺少配置，只提示用户提供以下最少必要信息，不要要求用户提供其他信息：
- `NOAH_MARKET_APIKEY`

不要要求用户提供 Base URL。Market 的 Base URL 已内置为 `https://securities-open-api.noahgroup.com`。

## 步骤 5：完成验证
运行最小验证命令，确认 market 可用。若缺 token，则只提示缺失项；否则继续，不要说额外废话。

安装脚本还会自动：
- 创建 `~/.openclaw/.secrets/`
- 生成 `noah-market.env` 模板（若不存在）
- 保留已有配置，不覆盖
- 检查 `NOAH_MARKET_APIKEY` 是否已填写
从 GitHub 仓库一键安装
git clone https://github.com/xuyun9160-lgtm/noah-agent-skills.git
cd noah-agent-skills
bash install_openclaw_skills.sh

Skills 功能一览

1. noah-stock-market — 市场行情助手

功能说明当前状态
市场快照获取股票最新报价、涨跌幅、成交量等已验证
K 线数据获取日 K、周 K、分钟 K，以及时间区间 K 线已验证
买卖盘获取实时买卖盘口挂单数据已验证
逐笔成交获取最近逐笔成交明细已验证
分时数据获取当日分时走势已验证
市场状态 / 全局状态查询市场开盘 / 休市状态及全局市场状态已验证
资金流向获取个股资金流向与分布已验证
基础信息 / 交易日历查询个股基础资料与交易日列表已验证
IPO 列表 / 排行榜新增 market 扩展能力已验证
条件选股 stock_filter协议已纳入，但当前环境不对外暴露暂不支持

2. noah-stock-trade — 交易模块（暂不默认安装）

noah-stock-trade 当前仍保留在仓库中继续完善，但本版本暂不作为默认安装项。后续如需单独启用，再额外开放安装。

使用方式

  • “查看腾讯的 K 线” — 自动调用市场查询
  • “看腾讯资金流向” — 自动调用资金流向查询
  • “看腾讯盘口” — 自动调用盘口查询
  • “看港股 IPO 列表” — 调用 ipo_list
  • “看港股涨跌幅排行榜” — 调用 rank

OpenClaw / CLI 使用示例：Market

python3 noah-stock-market/scripts/run_query.py snapshot HK-00700
python3 noah-stock-market/scripts/run_query.py kline HK-00700 num=10 ktype=K_DAY
python3 noah-stock-market/scripts/run_query.py capital_flow HK-00700 num=5
python3 noah-stock-market/scripts/run_query.py ipo_list HK market=HK
python3 noah-stock-market/scripts/run_query.py rank HK market_codes=HK rank_field=raisePercent page=1 page_size=10
